# Sales Data Analysis 2019

## Project Overview
This project analyzes 2019 sales data using Python to identify business insights such as monthly revenue, top-selling products, city-wise sales, and peak ordering hours.

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Features
- Data Cleaning
- Monthly Sales Analysis
- Revenue by Product
- Revenue by City
- Orders by Hour
- Most Ordered Products
- Heatmap Analysis
- Price vs Quantity Analysis

## Key Insights
- Highest revenue month identified.
- Best advertisement hour discovered.
- Highest revenue product and city analyzed.
- Accessories sell more units while electronics generate higher revenue.

## Dataset
Sales Data 2019 (12 Monthly CSV Files)
